# tarefa-1-cliente-servidor
Tarefa 1 desenvolvido para disciplina de Redes de Computadores. Esta aplicação tem a seguinte função: o usuário digita uma frase em caixa baixa na aplicação cliente, que envia uma requisição ao servidor. O servidor, ao receber a frase em caixa baixa, faz a sua conversão para caixa alta (com o método upper) e envia a frase modificada ao cliente.
